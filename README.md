# 🌿 Medicinal Plant Detection
A Flask-based web application that identifies medicinal plants from uploaded leaf images using a Convolutional Neural Network (CNN). The system predicts the plant species and provides basic medicinal information through a simple and user-friendly interface.

### Home Page
![Home Page](screenshots/home.png)

## Features
- User Registration & Login
- Plant Image Upload
- CNN-based Plant Classification
- Medicinal Plant Information
- Responsive Web Interface

## Tech Stack
- Python
- Flask
- TensorFlow & Keras
- HTML, CSS, JavaScript
- SQLite

## Installation
```bash
git clone https://github.com/Komal1817/medicinal-plant-detection.git
cd medicinal-plant-detection
pip install -r requirements.txt
python app.py
```

## Note
The trained model (`.h5`) and training dataset are not included in this repository due to their large size.
